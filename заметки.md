# Запись на стажировку

Прототип мини-аппа для записи официантов на стажировку.

## Что есть

- режим стажера;
- режим рекрута с демо-кодом доступа `2026`;
- создание дат стажировок;
- публичный список дат без зала и времени;
- запись стажера в очередь;
- общая очередь без даты для кандидатов без ограничений;
- отмена записи стажером, пока рекрут ее не обработал;
- подтверждение, перенос, отклонение и финальные статусы;
- форма стажера по старой логике из группы: ФИО, обучение, первая/повторная стажировка, опыт, ограничения;
- редактирование даты стажировки прямо в карточке рекрута;
- добавление кандидата из общей очереди в конкретную дату;
- отметки отчета кандидата и фидбека наставника;
- автоматическое закрытие даты, когда подтвержденных людей стало столько же, сколько мест;
- архив закрытых и прошедших дат внизу кабинета рекрута;
- хранение данных в `localStorage`.

## Как проверить

Открой `index.html` в браузере.

1. В режиме стажера введи ФИО, контакт и ограничения.
2. Выбери дату и нажми `Занять очередь`.
3. Перейди в режим рекрута.
4. Введи код `2026`.
5. Проверь новую заявку в разделе `Кандидаты`.
6. Подтверди кандидата или отправь его на другую дату.

## Для Пазл Бота

Если код вставляется в HTML-блок конструктора, используй `puzzlebot-embed.html`.

В нем нет `<!doctype html>`, `<html>`, `<head>` и `<body>`, а все стили привязаны к контейнеру `#internshipBookingApp`, чтобы конструктор меньше ломал внешний вид.
