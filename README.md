# Сервер запись на стажировку

Отдельный проект мини-приложения для записи официантов на стажировку.

## Файлы

- `index.html` - основная версия для браузера.
- `puzzlebot-embed.html` - версия для вставки в HTML-блок Пазл Бота.
- `заметки.md` - рабочие заметки по логике проекта.
- `server.js`, `package.json` - backend и зависимости.
- `Dockerfile`, `docker-compose.yml` - запуск проекта через Docker.
- `deploy.sh` - обновление проекта на сервере через GitHub.
- `DEPLOY.md` - инструкция по запуску на сервере.

## Текущая логика

- стажер заполняет данные и выбирает дату или общую очередь;
- рекрут входит по коду доступа;
- рекрут создает даты, смотрит кандидатов, подтверждает выход и меняет статусы;
- даты и заявки хранятся на сервере в `data/db.json`;
- профиль стажера хранится локально в браузере, чтобы не вводить ФИО заново.

## Быстрый запуск через Docker

```bash
docker compose up -d --build
```

После запуска проект будет доступен на порту `3000`.

## Обновление на сервере через GitHub

После подключения репозитория:

```bash
cd ~/ravshan/service-internship-booking
bash deploy.sh
```

Файл `data/db.json` не попадает в GitHub и остается на сервере.
